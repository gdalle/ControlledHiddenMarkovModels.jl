HiddenMarkovModels.DiscreteMarkovChainPrior — Type
DiscreteMarkovChainPrior

Define a Dirichlet prior on the initial distribution and on the transition matrix of a DiscreteMarkovChain.

Fields

  • π0_α::AbstractVector: Dirichlet parameter for the initial distribution
  • P_α::AbstractMatrix: Dirichlet parameters for the transition matrix

HiddenMarkovModels.DiscreteMarkovChainStats — Type
DiscreteMarkovChainStats

Store sufficient statistics for the likelihood of a DiscreteMarkovChain sample.

Fields

  • initialization_count::AbstractVector: count initializations in each state
  • transition_count::AbstractMatrix: count transitions between each pair of states

HiddenMarkovModels.HiddenMarkovModel — Type
HiddenMarkovModel{T,E}

Hidden Markov Model with arbitrary transition model (must be a discrete Markov chain) and emission distributions.

Fields

  • transitions::T: state evolution process.
  • emissions::Vector{E}: one emission distribution per state.
source

Prior for a HiddenMarkovModel.

Fields

  • transitions_prior::T: prior on the transition structure.
  • emissions_prior::Vector{E}: one prior per state emission distribution.
