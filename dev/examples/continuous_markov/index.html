Continuous Markov chain

using HiddenMarkovModels
using HiddenMarkovModels.LogarithmicNumbers
using Plots
using Random
using Statistics
rng = Random.default_rng()
Random.seed!(rng, 63)
Random.TaskLocalRNG()

Construction

A ContinuousMarkovChain object is built by combining a vector of initial probabilities with a matrix of transition rates.

p0 = [0.3, 0.7]
Q = [-2. 2.; 1. -1.]
mc = ContinuousMarkovChain(p0, Q)
ContinuousMarkovChain{Float64, Float64}([0.3, 0.7], [-2.0 2.0; 1.0 -1.0])

Simulation

To simulate it, we only need to decide the time interval.

h = rand(rng, mc, 4.2, 420.)
History{Int64,Float64} with 535 events on interval [4.2, 420.0)

Learning

Based on a history with integer marks, we can fit a ContinuousMarkovChain with Maximum Likelihood Estimation (MLE).

mc_mle = fit_mle(ContinuousMarkovChain{Float32, Float32}, h)


error_mle = mean(abs, intensity_matrix(mc_mle) - intensity_matrix(mc))
0.08545351028442383
