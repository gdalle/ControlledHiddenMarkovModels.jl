Multivariate Poisson process

using HiddenMarkovModels
using LogarithmicNumbers
using Plots
using Statistics

Construction

A MultivariatePoissonProcess object is built from a vector of positive event rates.

λ = rand(5)
pp = MultivariatePoissonProcess(λ)
MultivariatePoissonProcess{Float64}([0.5403668762836116, 0.37299325882612056, 0.5752273055396306, 0.9987345627250377, 0.3516593102737865])

Simulation

Since it is a temporal point process, we can simulate it on an arbitrary real interval.

history = rand(pp, 3.14, 314.0)
History{Int64,Float64} with 894 events on interval [3.14, 314.0)

Each event is defined by a time and an integer mark, which means we can visualize the history in 2 dimensions:

scatter(
    event_times(history),
    event_marks(history);
    title="Event history",
    xlabel="Time",
    ylabel="Mark",
    label=nothing,
    margin=5Plots.mm
)

Learning

Parameters can learned with Maximum Likelihood Estimation (MLE):

pp_est = fit_mle(MultivariatePoissonProcess{Float32}, history)
MultivariatePoissonProcess{Float32}(Float32[0.6112076, 0.37637523, 0.5790388, 0.9811491, 0.328122])

Let's see how well we did

error = mean(abs, pp_est.λ - pp.λ)
0.02383139927049043
